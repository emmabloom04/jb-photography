Overview

Purpose

To display Josh's photography portfolio for potential customers. To list pricing, information about Josh, and how to contact him. The main purpose of this site is to help Josh get more customers.

Audience

The audience is potential customers for Josh's photography.

Dynamic elements

Javascript will be used to create a picture slideshow on the homepage of the website. It will also be used to store the information about various pictures, and load all of them on to the website. There will also be tags associated with each picture, so that images can be searched for. When something is searched for, the array of picture objects will be filtered to only show the desired results.
